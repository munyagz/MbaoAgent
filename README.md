# MbaoAgent
This is a NodeJs project That gets data from one database and maps it into another

## Modules Used
* mysql
* mysql2 - To use Promises
* Dotenv - For enviroment variables

### To Run it
1. Clone the Repo
2. Run `npm install`
3. Run `npm run start`

### Settings
The settings are in the .env file and they include:
* DB_HOST=localhost
* DB_USER
* DB_PASSWORD
* DB
* DB2
* WAIT_TIME
* RECORD_LIMIT
